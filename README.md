# HR Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition patterns using Python and Power BI. The objective is to identify the major factors contributing to employee turnover and provide actionable business recommendations.

---

## Business Problem
A mid-sized IT services company is experiencing high employee attrition. This project aims to:

- Identify key drivers of employee turnover
- Analyze attrition patterns across departments
- Study the impact of salary, overtime, and performance
- Recommend retention strategies

---

## Tools & Technologies Used

- Python
- Pandas
- Matplotlib

- Power BI

## Dataset Information

Dataset contains:

- Employee ID
- Age
- Gender
- Department
- Salary
- Experience
- Performance Score
- Overtime
- Attrition

Total Records: 200

---

## Data Cleaning Steps

- Removed duplicate records
- Handled missing values
- Performed median and mode imputation
- Converted attrition column into numeric format
- Validated data types

## Key Insights

- Employees working overtime have significantly higher attrition
- Lower salary employees show slightly higher turnover
- Finance and IT departments have higher attrition rates
- Workload appears to influence employee retention

---

## Business Recommendations

- Reduce excessive overtime
- Improve workforce planning
- Introduce employee wellness initiatives
- Review compensation for lower salary bands
- Conduct employee engagement programs
